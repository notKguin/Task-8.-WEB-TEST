# Ручное тестирование (чек‑лист)

Дата проведения тестирования: 2026-01-11

Тестирование выполнено вручную в браузере Google Chrome с использованием инструментов разработчика.
Отметки в чек-листе проставлены на основании фактически выполненных проверок в локальной среде.

## Функциональность

| Проверка | Шаги | Ожидаемый результат | Статус |
|---|---|---|---|
| Нет «битых» ссылок | Проверка меню, кнопок, переходов между страницами | Ошибки 404 отсутствуют | ☑ |
| Список событий | Открытие главной страницы `/` | Список событий отображается | ☑ |
| Детальная события | Переход на `/events/<id>/` | Отображаются данные события | ☑ |
| Подача заявки | Авторизация → отправка формы заявки | Заявка создаётся, выполняется редирект | ☑ |
| Повторная заявка | Повторная отправка формы | Дубликат не создаётся | ☑ |
| Лайк | Нажатие кнопки лайка | Лайк добавляется и удаляется | ☑ |

## Формы и валидации

| Проверка | Шаги | Ожидаемый результат | Статус |
|---|---|---|---|
| Регистрация: обязательные поля | Отправка пустой формы | Отображаются ошибки | ☑ |
| Регистрация: пароль/повтор | Ввод разных паролей | Ошибка валидации | ☑ |
| Логин: неверные данные | Неверный пароль | Вход не выполняется | ☑ |
| Заявка: пустая мотивация | Отправка пустого поля | Ошибка, сохранения нет | ☑ |
